# Project Brief

## Identitas Proyek

Judul dosen: **Sistem Analisis Serangan DoS pada Arsitektur IoT**

Tema: **IoT + Cyber Security + Digital Forensics**

Dataset utama: **BoT-IoT (UNSW)**

Dataset alternatif: **RT-IoT2022 (UCI)**

## Judul Kerja

**Sistem Analisis Serangan DoS pada Trafik IoT Berbasis Machine Learning dan Network Forensics Menggunakan Dataset BoT-IoT**

## Latar Fokus

Proyek ini diarahkan sebagai eksperimen analisis trafik IoT berbasis dataset, bukan pembuatan hardware. Fokus utama adalah mendeteksi dan menganalisis serangan DoS/DDoS pada lingkungan IoT menggunakan pendekatan klasifikasi *machine learning* dan interpretasi fitur untuk mendukung *network forensics*.

## Scope Utama

1. Menggunakan BoT-IoT sebagai dataset utama.
2. Memulai dari klasifikasi biner: normal vs DoS/DDoS.
3. Membandingkan beberapa model ML klasik.
4. Menyimpan metrik evaluasi dan visualisasi.
5. Menganalisis fitur penting sebagai bagian dari forensik trafik.
6. Menyajikan hasil dalam naskah ilmiah dan dashboard static.

## Scope Tambahan

1. Klasifikasi multi-kelas.
2. Dataset RT-IoT2022 sebagai pembanding.
3. XGBoost/LightGBM.
4. SHAP atau explainability lanjutan.
5. GitHub Pages dashboard final.

## Output Akhir

- Naskah ilmiah.
- Notebook/script eksperimen.
- Tabel evaluasi.
- Grafik hasil.
- Analisis forensik fitur.
- Dashboard static.
- README dan dokumentasi reproducibility.
